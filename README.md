# Christian Paul Rodriguez Perez - Portafolio Web

## 📋 Descripción del Proyecto

Este es un proyecto final del curso **Tech-Challenge** que demuestra los conocimientos en HTML, CSS y control de versiones con Git y GitHub. La página web presenta un currículum profesional con un diseño moderno, responsivo y bien estructurado.

## ✨ Características

- ✅ **Estructura HTML Semántica**: Utiliza etiquetas semánticas como `<header>`, `<main>`, `<section>` y `<footer>`
- ✅ **Diseño Responsivo**: Adaptable a dispositivos móviles y de escritorio
- ✅ **Estilos CSS Profesionales**: Gradientes, animaciones y diseño moderno
- ✅ **Sección de Perfil**: Imagen de perfil y enlaces a redes profesionales
- ✅ **Enlaces Sociales**: GitHub, LinkedIn y WhatsApp integrados
- ✅ **Control de Versiones**: Gestión con Git
- ✅ **Publicado en GitHub Pages**: Accesible en línea

## 🚀 Requisitos Cumplidos

- [x] Etiquetas HTML semánticas (html, head, body, title, meta)
- [x] Títulos con jerarquía (h1, h2, etc.)
- [x] Párrafos descriptivos (p)
- [x] Enlaces (a) a recursos externos
- [x] Imágenes (img) con atributo alt
- [x] Listas ordenadas y desordenadas (ul, ol)
- [x] Control de versiones con Git
- [x] Repositorio en GitHub
- [x] Archivo README.md descriptivo
- [x] Publicado en GitHub Pages
